# Документація Charivna Craft

`docs/` — єдине джерело актуального плану, проєктного контексту, дорожніх карт і робочих рішень для Charivna Craft. Документи потрібні, щоб нові робочі чати ChatGPT могли стартувати без повторного переказування всієї історії проєкту.

## Індекс документів

| Файл | Призначення |
| --- | --- |
| [`MASTER_PLAN.md`](./MASTER_PLAN.md) | Загальний план, стратегічні цілі, кроспроєктні пріоритети й залежності. |
| [`TECHNICAL_ROADMAP.md`](./TECHNICAL_ROADMAP.md) | Технічний контекст, архітектура, backlog розробки, SEO-технічні задачі, тести й CI. |
| [`SEO_BLOG_ROADMAP.md`](./SEO_BLOG_ROADMAP.md) | SEO-стратегія блогу, keyword research, briefs, контент-план, перелінковка й вимірювання. |
| [`DESIGN_CONTENT_ROADMAP.md`](./DESIGN_CONTENT_ROADMAP.md) | Дизайн, візуальний контент, шаблон статті, фото товарів і UX-задачі. |
| [`DECISIONS.md`](./DECISIONS.md) | Журнал важливих рішень, які впливають на продукт, SEO, дизайн і workflow. |
| [`WORKFLOW.md`](./WORKFLOW.md) | Процес роботи в нових чатах, задачах, PR, документації та конфліктах між напрямами. |

## Статуси

* ✅ Завершено — задача виконана, змінена в PR, перевірена й може вважатися закритою після merge/production verification.
* 🟡 У роботі — задача активна, але ще не завершена.
* ⏭ Наступне — найближчий погоджений або рекомендований пріоритет.
* 🧊 Відкладено — задача зафіксована, але не входить у найближчий scope.
* 🔍 Потребує дослідження — потрібні додаткові дані, аналіз або рішення перед реалізацією.
